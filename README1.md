# PostgreSQL Relational Database Notes

This repository contains my learning notes, SQL practice queries, and hands-on projects while studying relational databases using PostgreSQL.

The primary project in this repository focuses on building a **Video Game Character Database** inspired by the FreeCodeCamp Relational Database curriculum.

---

# Skills Covered

- PostgreSQL setup and configuration
- Using the psql terminal
- Creating databases and tables
- SQL data types
- CRUD operations
- Primary keys and foreign keys
- Database relationships
- SQL joins
- Constraints and normalization
- Query optimization basics
- Transactions and indexing
